# CampusHUB
CampusHub offers you soo much more than you can expect :1. Previous Years Papers and Notes, Internship and Oppertunity board and soo much more to explore... 
CampusHub is a full-stack platform for college students.

## Planned Features

- Notes Sharing
- Campus-Annoucements
- Current council
- Clubs and student project(main idea and core- members)
- Internship Board
- User Profiles

Backend:
